# Лабиринт искр

Небольшая 2D-игра на Python без внешних библиотек. Используется стандартный `tkinter`, поэтому проект можно запустить обычным Python 3.10+.

## Как запустить

```bash
python main.py
```

Запускать нужно из папки проекта.

## Управление

- `WASD` или стрелки — плавное движение.
- `Space` или `E` — автоатака текущим оружием без ручного наведения.
- `Q` — быстро сменить оружие.
- `G`, `I` или `Tab` — открыть инвентарь.
- В инвентаре: `1–3` — выбрать оружие, `4–9` — купить умение за опыт.
- `Shift` — рывок, если умение открыто.
- `F` — ярость, если умение открыто.
- `P` или `Esc` — пауза.
- `F1` — отладочный режим: FPS, хитбоксы, состояния врагов, радиус оружия и пути A*.
- `F2` — замедление времени для отладки.
- `F3` — обычная скорость.
- `R` — новый забег с первого уровня.
- `Enter` — старт / новый забег после поражения.

## Цель

В игре 10 уровней. На каждом уровне нужно собрать 5 искр, найти три цветных ключа — красный, синий и зелёный — и дойти до выхода. Выход показывает три цветных гнезда над порталом: собранный ключ подсвечивает своё гнездо, несобранный остаётся серым.

После выхода с уровней 1–9 игрок переходит на следующий процедурный уровень. После выхода с 10-го уровня забег считается пройденным. HP, броня, опыт, оружие и умения сохраняются между уровнями, но искры и ключи нужно собирать заново.

Когда все условия для портала выполнены, включается режим тревоги: радиус света игрока уменьшается, а на карте появляется дополнительная волна врагов. Это делает последний путь до выхода напряжённее.

## Геймплейные механики

### Инвентарь

Инвентарь открывается клавишей `G`, `I` или `Tab`. Слева показано оружие, которое есть у игрока, справа — умения. Меню специально сделано простым: все действия выполняются цифровыми клавишами, поэтому не нужна мышка.

### Оружие

Оружие не требует ручного прицеливания: игрок нажимает `Space` или `E`, а игра сама выбирает ближайшего подходящего врага в радиусе.

- **Клинок** — ближняя автоатака по ближайшему врагу рядом с игроком.
- **Сюрикен** — средняя дистанция, один бросок по ближайшему видимому врагу.
- **Искромёт** — дальняя атака. Устройство выдерживает максимум 3 успешных выстрела. Каждый выстрел тратит 1 собранную искру, после третьего выстрела искромёт сгорает и позже спавнится снова.

На карте лежит больше искр, чем нужно для выхода, потому что часть искр можно потратить как ресурс для искромёта. Потраченная искра возвращается в лабиринт как новый предмет, чтобы игрок не попадал в безвыходную ситуацию.

### Опыт и умения

После убийства врагов выпадают фиолетовые сферы опыта. Опыт подбирается как обычный предмет и тратится в инвентаре на прокачку.

Умения:

- **Скорость** — пассивно увеличивает скорость героя.
- **Сила** — увеличивает урон оружия.
- **Живучесть** — повышает максимальное HP.
- **Рывок** — активное умение на `Shift`, позволяет быстро проскочить опасный участок.
- **Ярость** — активное умение на `F`, на несколько секунд увеличивает скорость и урон.
- **Магнит искр** — подтягивает искры и опыт с большего расстояния.

### Бусты

- **Аптечка** восстанавливает часть HP, но не выше максимума.
- **Броня** даёт запас защиты. При атаке врага броня сначала поглощает урон, и только остаток снимает HP.

### Враги и уровни

Враги патрулируют уровень, замечают игрока по радиусу и лучу видимости, затем преследуют через A*. Если игрок на прямой видимости, враг идёт напрямую; если за стеной — строит путь.

Типы врагов:

- **Страж** — обычный преследователь.
- **Стрелок** — держит дистанцию и выпускает снаряды.
- **Танк** — медленнее обычного врага, но имеет больше HP и наносит больше урона.

Сложность растёт от 1-го к 10-му уровню: скорость и урон врагов плавно увеличиваются. На первом уровне стражи заметно спокойнее, а к десятому становятся быстрыми и опасными.

### Движение и визуал

Движение героя сглажено: ввод задаёт целевую скорость, а персонаж постепенно набирает и сбрасывает скорость. Для отрисовки используется отдельная визуальная позиция `view_x/view_y`, поэтому движение выглядит мягче между фиксированными шагами логики.

Модель героя заменена с простого квадрата на маленького персонажа в капюшоне с плащом, лицом, глазами, шарфом, светящейся деталью и анимированными шагами. Враги тоже отличаются визуально: обычные стражи, стрелки и танки читаются разными силуэтами и цветами.

## Что реализовано по требованиям документации

- ТЗ вынесено в `TZ.md`.
- Код разделён на Model / View / Controller: `world.py`, `level.py`, `entities.py` — модель; `render.py` — отображение; `game.py` и `Input` — управление.
- Нет “god object” на всю игру: логика вынесена в отдельные системы в `systems.py`.
- Игровой цикл использует accumulator и фиксированный шаг `FIXED_DT = 1/60`.
- Порядок систем в `Game._fixed`: Input → Enemies → Movement+Collision → Logic → Animation → Cleanup.
- Уровень — тайловая матрица.
- Процедурная генерация уровня через DFS/backtracking-лабиринт на крупной сетке: коридоры шириной 3 тайла, повороты встречаются реже, дополнительные проходы создают циклы, seed отображается в HUD.
- BFS валидирует связность уровня.
- A* строит путь врагов к игроку.
- Враги используют конечный автомат `PATROL / CHASE / ATTACK`.
- Проверка видимости — raycast по тайлам.
- Коллизии с тайлами, предметами и снарядами — AABB.
- Камера следует за игроком и отрисовывает только видимую часть уровня.
- Освещение затемняет дальние тайлы, предметы, частицы и врагов; после открытия портала радиус света уменьшается.
- Искры имеют тёплое свечение, золотую форму и белую четырёхконечную сердцевину.
- Боевая система не требует прицеливания.
- Есть инвентарь, система опыта, прокачка умений и активные спецприёмы.
- Событийная шина `Bus` используется для сообщений и частиц.
- Частицы сделаны через пул объектов.
- Есть встроенная отладка: FPS, хитбоксы, пути, состояния врагов и радиус активного оружия.

## Проверка

```bash
python -m compileall .
python -m unittest discover -s tests -v
```

Тесты проверяют связность уровня, форму лабиринта, широкие коридоры, длинные прямые участки, появление ключей, оружия и дополнительных искр, автоатаку, искромёт как расходующий искры ресурс, лечение, броню, рост сложности, победу на 10-м уровне, сглаженный набор скорости игрока, выпадение опыта, покупку умений, спецприёмы, новых врагов, снаряды стрелков и режим тревоги у портала.
